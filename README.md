# 🚀 AmbitionBox Job Analysis & Visualization

A Flask-based web application that analyzes and visualizes AmbitionBox job and company data. Users can filter records by location, industry, and rating, then view results either as an interactive data table or through insightful visualizations.

## 📌 Project Overview

This project helps users explore job and company information collected from AmbitionBox. It provides filtering options and generates multiple charts for better data understanding and decision-making.

## ✨ Features

* 🔍 Filter data by:

  * Location
  * Industry
  * Rating

* 📊 Visualize data using:

  * Top Companies by Rating
  * Rating vs Number of Reviews
  * Industry Distribution
  * Rating Distribution
  * Average Rating by Industry
  * Jobs vs Salary Analysis

* 📋 View filtered data in tabular format

* 🎨 Modern and responsive UI using HTML, CSS, and Flask

## 📈 Generated Visualizations

### 1️⃣ Top Companies by Rating

Displays the highest-rated companies.

### 2️⃣ Rating vs Number of Reviews

Analyzes the relationship between company ratings and reviews.

### 3️⃣ Industry Distribution

Shows industry-wise company distribution using a pie chart.

### 4️⃣ Rating Distribution

Displays how ratings are distributed across companies.

### 5️⃣ Average Rating by Industry

Compares average ratings across industries.

### 6️⃣ Jobs vs Salary Analysis

Visualizes the relationship between available jobs and salary offerings.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/AmbitionBox-Job-Analysis.git
cd AmbitionBox-Job-Analysis
```

### Install Dependencies

```bash
pip install flask pandas matplotlib
```

### Run Application

```bash
python app.py
```

Open your browser and visit:

```bash
http://127.0.0.1:5700
```
